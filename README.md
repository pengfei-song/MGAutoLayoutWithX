## Installation

MGAutoLayoutWithX is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'MGAutoLayoutWithX'
```
##AppDelegate:

```ruby
MGAutoLayoutWithX.shared.isAutoLayout = true
```
